### Introduction
Welcome to the ***NexusByte AppStore***, a simple WebView application for downloading official and modified Android applications and games. The aim of this project is to build an open source store, without the need of an account or any of the tracking that other stores have (Google Play, Aptoide, HappyMod, etc). The store itself is a WebView. Why use WebView instead of develop a native application? A lot of people may not have experience with Android related languages. Web languages are relatively easy to work with and use, making WebView the best option for everyone.

---

### How It Works
The APK itself is built using Niotron. This repository is used for the front-end (pages, icons, listings, etc). Google Drive is used for actually storing the APK's (we will be swapping to an alternative storage host in the future).
